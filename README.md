# Dental Clinic Management System

Professional clinic operations software focused on patient care workflows, appointment scheduling, treatment tracking, billing, and reporting.

## Portfolio Summary

This project demonstrates end-to-end software delivery for a real business use case:
- Desktop-ready Python/Flask application architecture
- Structured patient and visit lifecycle management
- Appointment planning and reservation calendar workflows
- Billing, expense tracking, and operational reporting
- Packaging and deployment support for Windows distribution

## Tech Stack

- Python
- Flask
- SQLite
- HTML/CSS/JavaScript (web UI served by Flask)
